# Data Folder

This folder is intended to store raw sequencing data (FASTQ files).

Due to file size limitations, raw sequencing data is not included in this repository.

Example dataset used for testing:
SRR2584403 (available from ENA/SRA)

To download test data:

wget ftp://ftp.sra.ebi.ac.uk/vol1/fastq/SRR258/003/SRR2584403/SRR2584403_1.fastq.gz

wget ftp://ftp.sra.ebi.ac.uk/vol1/fastq/SRR258/003/SRR2584403/SRR2584403_2.fastq.gz
